Aula - 002

Converter "String" em inteiro

<script>
    
     //pedir para o usuário digitar o número
     var num1 = parseInt(prompt("Digite o primeiro número da soma:"));
    
     //pedir para o usuário digitar o número
     var num2 = parseInt(prompt("Digite o segundo número da soma:"));
    
     // mostrar a soma
     alert(num1 + num2);
</script>

Importante:

Lembrete:

Todo conteúdo (dado) enviado pelo usuário via interface, sempre iremos receber como uma "String", portanto nas situações em que estivermos trabalhando com números devemos sempre realizar conversões.

Dica de Ouro:

Para identificar o tipo do dado que estamos trabalhando use o comando 'typeof' seguido do nome da variável. Exemplo:
<script>
     // Identificar o tipo do dado que estamos trabalhando
     alert(typeof num1);
</script>

Resultado:
var valor = prompt("Digite um número:");
alert(typeof valor);

Obter dados de interação com usuário

Código Descrição
var nome = "Nome do usuário"; Declaração de variável String

prompt("Digite aqui ..."); Abre um insert na caixa de diálogo para interação do usuário
