Aula - 003

Operadores aritméticos - Prioridade entre tipos de operações

Símbolo Descrição
+ Adição (soma)
- Subtração
* Multiplicação
/ Divisão
% Módulo
** Exponenciação
++ Incremento
-- Decremento

Importante:

Os operadores "Multiplicação/Divisão" tem prioridade sobre os operadores "Soma/Subtração"

Lembrete:

Em situações onde há mais de uma operação aritmética acontecendo, deve-se atentar na prioridade de cada uma.
Exemplo: var media = n1 + n2 / 2;
Neste caso primeiro será realizado a divisão pois esta operação tem prioridade sobre a operação de soma.
Para corrigir use parenteses dessa forma:
var media = (n1 + n2) / 2;
Assim, primeiro são resolvidas as expressões mais encapsuladas até chegar no menor nível de encapsulamento.

<script>
    
     //pedir para o usuario digitar dois numero
     var num1 = parseFloat(prompt("Digite o primeiro número"));
    
     var num2 = parseFloat(prompt("Digite o segundo número"));
    
     var media = (num1 + num2) / 2;
    
     alert("A média é: " + media);
    
</script>

Aula - 002

Converter "String" em inteiro

<script>
    
     //pedir para o usuário digitar o número
     var num1 = parseInt(prompt("Digite o primeiro número da soma:"));
    
     //pedir para o usuário digitar o número
     var num2 = parseInt(prompt("Digite o segundo número da soma:"));
    
     // mostrar a soma
     alert(num1 + num2);
</script>
    

Importante:

Lembrete:

     1 - Todo conteúdo (dado) enviado pelo usuário via interface, sempre iremos receber como uma "String", portanto nas situações em que estivermos trabalhando com números devemos sempre realizar conversões para o tipo necessário.

     2 - Na conversão o comando "parseInt()" sempre descarta os valores decimais,
IMPORTENTE não é um arredondamento e sim o descarte dos decimais.

Dica de Ouro:

     Para identificar o tipo do dado que estamos trabalhando use o comando 'typeof' seguido do nome da variável. Exemplo:
<script> // Identificar o tipo do dado que estamos trabalhando
     var valor = prompt("Digite um número:");
     alert(typeof valor);
</script>

Resultado:

     var valor = prompt("Digite um número:");


     alert(typeof valor);
